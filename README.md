# Nestify Docs

Documentation site for **Nestify** CLI tool. Built with Next.js and Tailwind CSS.

## Quick Start

```bash
# Clone
git clone https://github.com/mugabodannyshafi/nestify-docs.git
cd nestify-docs

# Install
npm install

# Run
npm run dev
```

Open [http://localhost:3000](http://localhost:8080)

## Build

```bash
npm run build
npm run start
```
